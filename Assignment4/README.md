For this assignment, I wanted to create a small website about my cat Ares who passed last week. I want to talk about him and show photos of him to other people besides my family. I wanted the site to be personal, show the cute photos I have of him, while still showing that I understand how to use Flexbox and CSS Grid. I think I like Flexbox more so I can add context to some of the photos, but there will be times I will prefer Grid more so I can just the images off and they can speak for themselves.
